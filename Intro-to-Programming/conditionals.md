# Conditionals

## What are conditionals for?

* We use condtionals to make decisions in code

## Structure of an if statement

```ruby
num = 42
	
if num > 0
	puts "positive"
else 
	puts "negative"
end
```

```ruby
num = 42
	
if num > 0
	puts "positive"
elsif num < 0
	puts "negative"
else 
	puts "it is zero!"
end
```

## Else or els

* Else or els makes the conditionals exclusive
